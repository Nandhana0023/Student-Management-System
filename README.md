# Project Title

Title: Student Management System

Author: Nandhana MA

Description: A simple Command Line Interface (CLI) application in Java to perform CRUD operations (Create, Read, Update, Delete) on student records.


## Technologies Used
- Java

- VS Code 

- ArrayList for storing student records

- OOP Concepts (Class & Objects)
## Features
- Add Student – Store a new student with ID, name, and marks

- View Students – Display all saved student records

- Update Student – Modify a student’s name or marks using their ID

- Delete Student – Remove a student from the system by ID
